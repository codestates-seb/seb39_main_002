# My Todo App

seb39_main_002 프로젝트입니다.

## 프로젝트 설명
- 나만의 냉장고 관리: 장고관리
- 냉장고 속 재료들을 손쉽게 관리해주고 재료에 맞는 레시피를 추천

## 팀원 소개

- FE: 류지창, 이준호
- BE: 김문식, 정진철, 금도영

## 기능 소개
- 회원가입 기능
- 로그인 기능
- 메모 등록,삭제 기능
- 내정보 수정 기능
- 냉장고 재료 CRUD 기능
- 냉장실, 냉동실 재료 이동 기능
- 냉장고 재료에 따른 레시피 추천 기능
